# IonaCRM - A Simple & Straight CRM with Symfony 4

IonaCRM is a Customer Relationship Management tool that aims to help manage your business easily

## Features

Marketing
+ Contacts/ customers

Inventory
+ Products

Sales
+ Quotes
+ Invoices

System
+ Users
+ Configuration
+ Multi language support
